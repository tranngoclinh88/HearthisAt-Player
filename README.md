# Hearthis.at Player
Minimalistic iOS player for Hearthis.at top tracks and artists.

## About
**Xcode Version** 8.3.x  
**Deployment Target** iOS 10.0  
**Swift Version** 3.1

## Installation
- Clone the repo.
- Run `pod install` to install dependencies.
- Open and run the workspace.
